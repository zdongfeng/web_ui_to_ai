# Prompt: Extract Pattern From Reference

Use this prompt during research.

```text
Analyze the following product UI reference and extract reusable AIUI design decisions.

Reference:
<paste screenshot description, URL notes, or product notes>

Do not copy brand-specific style. Extract reusable structure.

Return:

1. Page type
2. User goal
3. Layout model
4. Regions
5. Component composition
6. Density
7. Color strategy
8. Typography character
9. Interaction model
10. Useful reusable decisions
11. What not to copy
12. AIUI pattern candidates
13. AIUI component candidates
14. Avoid rules

Then draft one AIUI pattern YAML if a reusable pattern is present.
```
