# Phase 1：设计研究

## 目标

从真实产品 pattern 中建立第一批设计知识库。

## 研究对象

可以参考优秀产品和设计系统，但不要复制它们的品牌身份。

建议研究：

- Linear
- Notion
- Vercel
- Stripe
- GitHub
- Raycast
- Figma
- Supabase
- Shopify
- Retool

## 研究模板

每个参考对象都要提取：

```text
Product type:
Primary users:
Register: product or brand
Layout model:
Navigation model:
Density:
Typography:
Color strategy:
Surface strategy:
Common components:
Common patterns:
Useful design decisions:
Anti-patterns to avoid:
```

## AI 的角色

AI 可以：

- 分析截图
- 总结视觉 pattern
- 对比多个产品
- 提议可复用 pattern 名称
- 起草研究笔记

## 人的角色

人必须：

- 拒绝表层风格复制
- 重命名 pattern
- 决定哪些进入 AIUI v0.1
- 选择第一批 style profiles

## 产出

- 10 份 reference notes
- 10 个 pattern candidates
- 20 个 component candidates
- 3 个 style profile candidates
- 初始 anti-pattern list

## 退出标准

团队有足够依据定义第一版 AIUI library，而不是凭空发明。
