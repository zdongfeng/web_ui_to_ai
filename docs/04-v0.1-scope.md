# AIUI v0.1 Scope

v0.1 should be small, testable, and useful.

## Must Have

### Spec

- Foundation principles
- Token model
- Component model
- Pattern model
- Page model
- Brand profile model

### Library

- 20 components
- 10 patterns
- 5 pages
- 3 brand profiles

### Agent Usage

- Agent handoff template
- React + Tailwind adapter
- 3 example page specs

### Validation

- Required field checks
- Reference checks
- Duplicate name checks
- Circular dependency checks
- Basic page completeness checks

## Should Have

- Clear naming convention
- Example project `.aiui/` folder
- Comparison prompt for AI benchmark
- Minimal CLI plan

## Not in v0.1

- Visual canvas
- Figma replacement
- Automatic screenshot-based design judging
- Full code generator
- Multi-framework generation
- Automatic web crawling
- Enterprise design system ingestion
- Large marketplace of styles

## First Supported User

The first user is:

```text
A developer building a product with AI coding agents who wants better frontend pages without spending time as a designer.
```

## First Supported Stack

```text
React + Tailwind
```

This does not mean AIUI is React-specific. It means the first adapter is React + Tailwind so the project can be validated quickly.
