# AIUI v0.1 范围

v0.1 必须小、可测试、可用。

## 必须包含

### Spec

- Foundation principles
- Token model
- Component model
- Pattern model
- Page model
- Brand profile model

### Library

- 20 个 components
- 10 个 patterns
- 5 个 pages
- 3 个 brand profiles

### Agent Usage

- Agent handoff template
- React + Tailwind adapter
- 3 个 page spec 示例

### Validation

- 必填字段检查
- 引用检查
- 重复命名检查
- 循环依赖检查
- 基础 page completeness 检查

## 应该包含

- 清晰命名规范
- 示例项目 `.aiui/` 文件夹
- AI benchmark 对比 prompt
- 最小 CLI 设计

## v0.1 不做

- 可视化画布
- Figma 替代品
- 自动截图审美评分
- 完整代码生成器
- 多框架生成
- 自动爬全网
- 企业设计系统自动导入
- 大型风格市场

## 第一批用户

```text
使用 AI coding agent 开发产品页面、希望前端更好但不想投入大量设计时间的开发者。
```

## 第一版技术栈

```text
React + Tailwind
```

这不代表 AIUI 绑定 React。它只代表第一版 adapter 选择 React + Tailwind，方便快速验证。
