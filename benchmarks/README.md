# Benchmarks

Benchmarks 用来证明 AIUI 是否真的改善 AI 生成前端的质量。

基本方法很简单：

```text
同一个页面需求
→ 不使用 AIUI 生成
→ 使用 AIUI 生成
→ 对比结果
```

## 第一批任务

1. Order management page
2. SaaS settings page
3. AI tool landing page

## 必须记录的内容

每个任务收集：

- Direct prompt
- AIUI-assisted prompt
- 生成结果截图或说明
- Human score
- Failure notes
- Spec improvement suggestions
